## HTML Editor
This is a simple HTML editor. You can edit your HTML code and save it to a file. You can also load a file and edit it

* [ ] ~~Syntax highlighting~~
* [x] Auto Tag Closing
* [x] Auto Syntax Completion
* [x] File Saving and Loading 
* [x] User Friendly
